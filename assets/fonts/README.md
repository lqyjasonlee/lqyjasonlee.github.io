# Local fonts

This site expects these Lato files in `assets/fonts/`:

- Lato-Light.woff
- Lato-Light.woff2
- Lato-LightItalic.woff
- Lato-LightItalic.woff2
- Lato-Regular.woff
- Lato-Regular.woff2
- Lato-Italic.woff
- Lato-Italic.woff2
- Lato-Bold.woff
- Lato-Bold.woff2
- Lato-BoldItalic.woff
- Lato-BoldItalic.woff2

You already have these files in your GitHub repo under `assets/fonts/`.
Copy them into this folder if you are moving the site to a fresh repository.
